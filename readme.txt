Hogy olyan körulmények között tudd futtatni, ahogy én, ahhoz kell egy MySql alatt futó localhost.
Én ezt XAMP -al oldottam meg. Ha elindítottad a mySql adatbázis kezelõt, utána a localhoston 
" http://localhost/phpmyadmin/" hozz létre egy új adatbázist "point" néven, majd ebbe belépve
 importáld be az általam küldött "points.sql" 
adatbázbázist.
A program megfelelõ mûködéséhez szükséges a mûködõ adatbázis.

A programot a "dyntellTree.jar" -al tudod elindítani.

Ha valami nem úgy megy ahogy kellene, írj nyugodtan :-)
